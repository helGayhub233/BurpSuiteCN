## 📦 快速开始

1. 在 Burp Suite 安装目录中找到并编辑 `vmoptions.txt`
2. 编辑现有条目或新增一行：

```
-javaagent:/绝对路径/burpsuiteloader.jar=loader,chs
```

3. 配置完成后，直接使用以下方式启动：
   - 双击 Burp Suite 应用
   - 启动脚本 / 快捷方式
